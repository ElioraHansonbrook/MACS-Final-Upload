# README

Thank you so much for taking a look at my project! It discusses the emerging war in Iran using a fun and exciting series of visualizations. It is best viewed [here](http://projects.hansonbrook.com). (Link not working? Go to projects.hansonbrook.com)

A full writeup is in the top level of this repository as "MACS Data Vis Final WriteUp.pdf"

## Compiling
Please note that if you want to compile the project yourself, you will need to provide your own set of shapefiles. To do this,
simply download shapefiles (from [GeoBoundaries](https://doi.org/10.1371/journal.pone.0231866), for example) for the relevant countries and rename the folder containing them "shapefiles".
Once this folder is renamed, simply place it at the top level of your clone of this repository, and all code should run as expected.
