# README

Thank you so much for taking a look at my project! It discusses the emerging war in Iran using a fun and exciting series of visualizations. It is best viewed [here](projects.hansonbrook.com).

## Compiling
Please note that if you want to compile the project yourself, you will need to provide your own set of shapefiles. To do this,
simply download shapefiles (from GeoBoundaries, for example) for the relevant countries and rename the folder containing them "shapefiles".
Once this folder is renamed, simply place it at the top level of your clone of this repository, and all code should run as expected.
